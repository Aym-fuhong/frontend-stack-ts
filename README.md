## Install brew

If your machine already installed yarn, please skip this step.

```shell
brew install yarn
```

## Install dependency packages

```shell
yarn
```

## Running

Run eslint

```shell
yarn eslint
```

Run eslint --fix

```shell
yarn eslint-fix
```

Start project

```shell
yarn start
```

Run test

```shell
yarn test
```
